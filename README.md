# Shanhe Gongfu · Rural Revitalization Case Library

A static website showcasing fieldwork results from Henan, Sichuan, and Yunnan. Includes interactive map, photo gallery, interview video, and downloadable PDF toolkits.

**Access locally:** Use Live Server or Python HTTP server on `index.html`.

**Deploy:** Upload to GitHub Pages for public access.
